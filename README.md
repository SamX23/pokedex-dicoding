# Simple Pokedex Using Webpack

<img height="600" src="https://github.com/SamX23/pokedex-dicoding/blob/master/screenshot/pokemon.png">

## How to use

Clone repo
'gh repo clone SamX23/simple-pokedex'

Install dependencies
'npm run install'

Start development
'npm run start'

Building apps
'npm run build'
